# Antibiosis-Game
Antibiosis: The Cost of Progress > An interactive strategy simulation built with HTML5 Canvas and JavaScript. Players manage the delicate balance between industrial expansion and environmental preservation, navigating economic tech tiers while mitigating real-time pollution dynamics.
